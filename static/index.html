BROWSERIFY JUST GOT MORE MAGICAL.

See those colors?^^ This is where they came from:

<!--All you need is a script tag!-->
<script src='http://browserify.nodejitsu.com/?require=["mrcolor", {"jquery": "jquery-browserify"}]'  type="text/javascript"></script>
<!--This is your entry point.-->
<script type="text/javascript">
  // Use require in the browser!
  var color = require("mrcolor")(),
      $ = require("jquery");

  // Now do some magic with your new-found powers! >:3
  $(function () {
    $(".rainbow").replaceWith((function () {
      var h = $("<h1>");
      $(".rainbow")
        .text()
        .split("")
        .forEach(function (c) {
          h.append($("<span>")
            .css("color", "rgb(" + color().rgb().join(",")+")")
            .text(c)
          );
        });

      return h.addClass("rainbow");
    })());
  });
</script>

HOW IT WORKS:

1. Use a querystring to specify the "requires" parameter of browserify's API:

require('browserify')({
      require : ["mrcolor", {"jquery": "jquery-browserify"}]
  }))

becomes:

http://browserify.nodejitsu.com/?require=["mrcolor", {"jquery": "jquery-browserify"}]

2. browserify.nodejitsu.com browserifies modules for you!

3. browserify.nodejitsu.com downloads your package from npm if necessary!

4. browserify.nodejitsu.com uses couchdb to cache browserify bundles so you don't have to wait!

BASICALLY IT'S PRETTY SWEET.
